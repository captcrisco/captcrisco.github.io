# captcrisco.github.io

Title: Intrusion Methodologies & Log Analysis

Description:
This project explores common intrusion techniques and how they appear in real-world system logs. I analyzed methods such as credential abuse, brute force attacks, and service misuse, and identified their indicators within Windows and Linux logs. The project also focuses on distinguishing normal system activity from suspicious behavior and documenting appropriate analyst responses. Through this work, I developed a stronger understanding of log analysis, threat detection, and security monitoring practices.

Tools & Skills:

Windows Event Viewer (Security/System Logs)
Linux log analysis (auth.log, syslog)
Log interpretation & event correlation
Threat detection & incident analysis
Understanding of MITRE ATT&CK techniques
Basic cybersecurity investigation methodology
Documentation & reporting skills
